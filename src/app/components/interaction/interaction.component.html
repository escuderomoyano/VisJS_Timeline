Este ejemplo muestra funciones para ajustar mediante programación la ventana visible de la línea de tiempo.










Este ejemplo muestra cómo usar la opción clickToUse: antes de que pueda desplazarse y arrastrar en la línea de tiempo, primero debe hacer clic en la línea de tiempo para activarla.


Este ejemplo escucha los eventos select, click, doubleClick, rangechange, y rangechanged del Timeline (otros possibles eventos: mouseDown, mouseUp, mouseOver, mouseMove), y escucha los cambios en el DataSet (add, update, o remove items).


El rango para el Timeline esta limitado de la siguiente manera:

  • La fecha mínima visible está limitada a 2018-01-01 usando la opción min
  • La fecha máxima visible está limitada a 2019-01-01 (excluida) usando la opción max
  • El zoom visible se limita a un mínimo de 24 horas usando la opción zoomMin
  • El zoom visible se limita a un máximo de aprox. 3 meses usando la opción zoomMax
